Surgimento da internet

A internet foi criada em 1969, nos Estados Unidos. Chamada de Arpanet,

tinha como função interligar laboratórios de pesquisa. Naquele ano,

um professor daUniversidade da Califórnia passou para um amigo em Stanford o primeiro e-mail da história.

Primeiros computadores

MARK 1

O Harvard Mark I, também chamado Harvard-IBM Automatic Sequence Controlled Calculator (Calculador controlado de sequências automáticas Harvard - IBM),

foi terminado em 1944 pelo engenheiro de computadores e matemático Howard H. Aiken

em colaboração com uma equipa de engenheiros da IBM, liderados por Clair D. Lake

Conheça o MARK 1 (Voltar)

ENIAC

Desenvolvido a pedido do exército dos Estados Unidos para seu laboratório de pesquisa balística,

o ENIAC era um monstrengo de 30 toneladas de peso que ocupava uma área de 180 m² de área construída.

Sua produção custou nada menos do que US$ 500 mil na época.

A máquina contava com um hardware equipado com 70 mil resístores e 18 mil válvulas de vácuo

que em funcionamento consumiam vorazmente 200 mil watts de energia.

Conheça o ENIAC (Voltar)

COLOSSUS

A máquina ocupa uma sala inteira, é formada por várias peças aposentadas

há décadas e nem contava com recursos como tela ou memória.

Mas ele surgiu anos antes do ENIAC, em janeiro de 1944,e tinha uma função ainda mais nobre

do que fazer cálculos: decodificar mensagens trocadas entre soldados nazistas,

permitindo que os Aliados antecipassem e neutralizassem os movimentos inimigos.

Historia sobre HTML

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web.

Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas

  • (CSS) para descrever a aparência/apresentação
  • (JavaScript) funcionalidade/comportamento

Css e Javascript auxiliam na estilização de um site.

Client X Sever

Cliente ou cliente é a parte onde o cliente tem interação/visualização,

por conta disso buscamos deixar de modo claro e interativo para o usuario.

Sever ou servidor é onde esta as informações onde o usuario não tem acesso visual.

(Voltar)

Sever-SIde

Sever ou servidor é o local onde ocorre o armazenamento das informações, por exemplo:

  • Arquivos
  • Banco de dados
  • Streaming
  • E-mail
  • Servidor Proxy

Servidor voltado a arquivos

Quando temos um servidor com o foco em ARQUIVOS temos uma gama de possibilidade para armazenamento

Por exemplo Imagens, Videos, Musicas e etc..

qualquer computador pode ser configurado como um file server,

porém equipamentos para esse fim como storages NAS cumprem muito melhor essa função.

Servidor voltado a segurança

Alguns equipamentos são configurados para potencializar e dar eficiência, de forma dedicada,

ao processamento e a transferência de dados entre os demais sistemas computacionais da infraestrutura de TI,

como alguns servidores de aplicação e storages.

Servidor voltado a Streaming

Esses equipamentos podem transmitir conteúdo de áudio ou vídeo via internet, através de um processo conhecido como streaming.

podemos utilizar como exemplo:Netflix, Amazon Prime, Youtube e outras.

Nessa categoria podemos inserir até as IPTV, Pois atraves da internet temos acesso a uma diversidade de conteúdo.

Servidor voltado a E-mail

Servidores de email são equipamentos configurados para armazenar e transferir

e-mails através de redes locais ou via internet.

Servidor Proxy

Atuando como mediador entre as requisições dos computadores clientes que buscam recursos de outros servidores,

como arquivos, páginas web e outros serviços, o servidor proxy filtra as solicitações.

Esse servidor é, por exemplo, intermediário entre um acesso feito por uma estação de trabalho a um servidor web.

Client-Side

As linguagens client-side são linguagens onde apenas o seu NAVEGADOR vai entender.

Quem vai processar essa linguagem não é o servidor, mas o seu browser.

Por exemplo: se criarmos um script em linguagem back-end (PHP, Asp, Java, Ruby, etc) que apenas calcula a soma de 2 + 2,

será o SERVIDOR (ou back, o server) que calculará este resultado.
